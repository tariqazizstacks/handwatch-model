# handwatch-model
This project is a digital handwatch model built using HTML, CSS, and JavaScript. It simulates a modern wristwatch design with a real-time digital clock displayed inside the dial.

****Features****
Real-time clock updates every second using JavaScript.
Responsive design that adapts to desktop and mobile screens.
Customizable watch background (PNG image of a handwatch).
Stylish rectangular dial with bold digital time display of ferrari design.
Easy to integrate into any webpage or project.
